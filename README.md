Tested on Windows 8

# Requirements

- python >=3.6
- pip >=20.3

# Installation

Run `pip install -r requirements.txt`

# Usage

1. Download the assignment archive from Moodle
2. Run `main.py <path_to_archive>.zip`
3. Go to `<path_to_archive>` and check HTML and CSS errors in each `assignments/<student>.log`
